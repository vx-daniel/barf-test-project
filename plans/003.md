# Plan: Mode Calculation

## Overview
Implement a `mode(nums)` function that finds the most frequently occurring value(s) in a number array.

## Approach
1. Build a frequency map using `Map<number, number>`
2. Find the maximum frequency
3. Collect all values that match the maximum frequency
4. Return as a sorted array

## Implementation
- Add `mode` function to `src/index.ts`
- Create `tests/mode.test.ts`

## Edge Cases
- Single element array: return that element
- All elements same frequency: return all elements
- Empty array: throw
- Negative numbers and decimals
