# Plan: Standard Deviation

## Overview
Implement standard deviation calculation supporting both population and sample variants.

## Approach
1. Reuse the existing `mean()` function from the library
2. Compute sum of squared deviations from mean
3. Divide by N (population) or N-1 (sample)
4. Return square root of variance

## Implementation
- Add `stddev` function to `src/index.ts`
- Signature: `stddev(nums: number[], opts?: { sample?: boolean }): number`
- Create `tests/stddev.test.ts`

## Edge Cases
- Empty array: throw
- Single element with sample=true: throw (division by zero in N-1)
- All identical values: return 0
