Add an `isPalindrome` function to `src/index.ts`.

## Function

**`isPalindrome(s: string): boolean`**
Returns `true` if the string reads the same forwards and backwards (case-insensitive, ignoring non-alphanumeric characters).

Examples:
```
isPalindrome('racecar')       → true
isPalindrome('A man a plan a canal Panama')  → true
isPalindrome('hello')         → false
isPalindrome('')              → true
```

## Acceptance Criteria

- [ ] Returns `true` for classic palindromes like `'racecar'`
- [ ] Comparison is case-insensitive
- [ ] Non-alphanumeric characters are ignored
- [ ] `isPalindrome('')` returns `true`
- [ ] Returns `false` for non-palindromes
- [ ] Exported from `src/index.ts`
- [ ] Tests in `tests/isPalindrome.test.ts`
