## Description
Implement various rounding strategies beyond Math.round.

## Acceptance Criteria
- [ ] `roundTo(n, decimals)` rounds to N decimal places
- [ ] `ceil(n, decimals?)` ceiling to N decimal places
- [ ] `floor(n, decimals?)` floor to N decimal places
- [ ] `roundHalfEven(n)` banker's rounding
- [ ] Tests cover each rounding mode and floating-point edge cases
