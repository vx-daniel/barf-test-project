Add a `kebabCase` function to `src/index.ts`.

## Function

**`kebabCase(s: string): string`**
Converts a string to kebab-case. Splits on whitespace, hyphens, underscores, and camelCase boundaries; joins with `-`; lowercases all words.

Examples:
```
kebabCase('Hello World')   → 'hello-world'
kebabCase('fooBarBaz')     → 'foo-bar-baz'
kebabCase('foo_bar')       → 'foo-bar'
kebabCase('  hello  ')     → 'hello'
```

## Acceptance Criteria

- [ ] Splits on spaces, underscores, and existing hyphens
- [ ] Inserts `-` between camelCase word boundaries
- [ ] All output characters are lowercase
