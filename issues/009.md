Add a `camelCase` function to `src/index.ts`.

## Function

**`camelCase(s: string): string`**
Converts a string to camelCase. Splits on whitespace, hyphens, and underscores.

Examples:
```
camelCase('hello world')    → 'helloWorld'
camelCase('foo-bar-baz')    → 'fooBarBaz'
camelCase('foo_bar')        → 'fooBar'
camelCase('  hello  world') → 'helloWorld'
```

## Acceptance Criteria

- [ ] Splits on spaces, hyphens, and underscores
- [ ] First word is all lowercase
- [ ] Subsequent words are capitalized
- [ ] Leading/trailing delimiters are ignored
- [ ] `camelCase('')` returns `''`
- [ ] Exported from `src/index.ts`
