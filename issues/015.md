Add a `template` function to `src/index.ts` that performs simple `{{key}}` interpolation on a string.

## Function

**`template(s: string, vars: Record<string, string>): string`**
Replaces all `{{key}}` placeholders with the corresponding value from `vars`. Missing keys leave the placeholder unchanged.

Examples:
```
template('Hello, {{name}}!', { name: 'world' })  → 'Hello, world!'
template('{{a}} + {{b}}', { a: '1' })             → '1 + {{b}}'
```

## Acceptance Criteria

- [ ] Replaces `{{key}}` placeholders with matching values
- [ ] Missing keys are left as-is
- [ ] Handles multiple placeholders in one string
- [ ] Handles the same placeholder appearing more than once
- [ ] `template('', {})` returns `''`
- [ ] Exported from `src/index.ts`
- [ ] Tests in `tests/template.test.ts`

## Blocker

Unclear whether placeholders with whitespace (e.g. `{{ name }}`) should be supported. Also need a decision on escaping — how to produce a literal `{{` in the output. Blocked pending requirements clarification.
