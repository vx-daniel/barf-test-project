## Description
Implement an `isPrime(n)` function that checks whether a number is prime.

## Acceptance Criteria
- [ ] `isPrime(n)` returns true for prime numbers, false otherwise
- [ ] Handles edge cases: 0, 1, negative numbers
- [ ] Efficient for large numbers (trial division up to sqrt)
- [ ] Tests cover primes, composites, and edge cases
