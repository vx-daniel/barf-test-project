Add a `reverse` function to `src/index.ts`.

## Function

**`reverse(s: string): string`**
Returns the string with characters in reverse order.

Examples:
```
reverse('hello')   → 'olleh'
reverse('abcd')    → 'dcba'
reverse('')        → ''
reverse('a')       → 'a'
```

## Acceptance Criteria

- [ ] Returns characters in reverse order
- [ ] `reverse('')` returns `''`
- [ ] `reverse('a')` returns `'a'`
- [ ] Handles strings with spaces correctly
