## Description
Implement the `formatWithCommas(n)` number formatting utility that adds thousand separators to a number string, and write its tests.

## Acceptance Criteria
- [x] `formatWithCommas(n)` adds thousand separators (e.g. `1234567` → `"1,234,567"`)
- [x] Handles negative numbers (e.g. `formatWithCommas(-1234)` → `"-1,234"`)
- [x] Preserves decimal part unchanged (e.g. `formatWithCommas(1234.5678)` → `"1,234.5678"`)
- [x] Returns `"0"` for zero
- [x] Handles very large numbers (e.g. `formatWithCommas(1e15)` → `"1,000,000,000,000,000"`)
- [x] Returns `"NaN"` for `NaN`, `"Infinity"` / `"-Infinity"` for ±Infinity
- [x] Exported from `src/index.ts`
- [x] Tests pass in `tests/formatting.test.ts`

## Implementation Notes
- Guard NaN/±Infinity → return string literal
- Split number string on `.` to separate integer and decimal parts
- Handle negative sign separately
- Insert commas every 3 digits from right in integer part
- Rejoin with decimal part if present
