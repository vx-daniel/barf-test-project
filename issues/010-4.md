## Description
Implement the `toScientific(n, precision?)` number formatting utility that formats a number in scientific notation, and write its tests.

## Acceptance Criteria
- [x] `toScientific(n, precision?)` formats a number in scientific notation (e.g. `1234` → `"1.234e+3"`)
- [x] `precision` is optional; omitting it uses JS default of 6 significant digits via `toExponential()`
- [x] Supports custom precision (e.g. `toScientific(1234, 2)` → `"1.23e+3"`)
- [x] Returns `"0e+0"` for zero
- [x] Handles negative numbers (e.g. `toScientific(-1234)` → `"-1.234e+3"`)
- [x] Returns `"NaN"`, `"Infinity"`, `"-Infinity"` for those special values
- [x] Exported from `src/index.ts`
- [x] Tests pass in `tests/formatting.test.ts`

## Implementation Notes
- Guard NaN/±Infinity → return string literal
- Call `n.toExponential(precision)` — omit the argument entirely if `precision` is `undefined`
