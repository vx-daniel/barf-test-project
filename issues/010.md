Add a `snakeCase` function to `src/index.ts`.

## Function

**`snakeCase(s: string): string`**
Converts a string to snake_case. Splits on whitespace, hyphens, and underscores; joins with `_`; lowercases all words.

Examples:
```
snakeCase('Hello World')   → 'hello_world'
snakeCase('fooBarBaz')     → 'foo_bar_baz'
snakeCase('foo-bar')       → 'foo_bar'
snakeCase('  hello  ')     → 'hello'
```

## Acceptance Criteria

- [ ] Splits on spaces, hyphens, and underscores
- [ ] Inserts `_` between camelCase word boundaries
- [ ] All output characters are lowercase
- [ ] Leading/trailing delimiters are ignored
