## Description
Implement the `formatPercent(n, decimals?)` number formatting utility that formats a ratio as a percentage string, and write its tests.

## Acceptance Criteria
- [ ] `formatPercent(n, decimals?)` formats a ratio as a percentage (e.g. `0.1234` → `"12.34%"`)
- [ ] `decimals` defaults to `2` when omitted
- [ ] Supports custom decimal places (e.g. `formatPercent(0.126, 0)` → `"13%"`)
- [ ] Returns `"0.00%"` for zero
- [ ] Handles negative numbers (e.g. `formatPercent(-0.1234)` → `"-12.34%"`)
- [ ] Returns `"NaN%"` for `NaN`, `"Infinity%"` / `"-Infinity%"` (or string literals) for ±Infinity
- [ ] Exported from `src/index.ts`
- [ ] Tests pass in `tests/formatting.test.ts`

## Implementation Notes
- Guard NaN/±Infinity → return string literal
- Multiply input by 100, call `toFixed(decimals ?? 2)`, append `"%"`
