## Description
Implement the `formatPercent(n, decimals?)` number formatting utility that formats a ratio as a percentage string, and write its tests.

## Acceptance Criteria
- [x] `formatPercent(n, decimals?)` formats a ratio as a percentage (e.g. `0.1234` → `"12.34%"`)
- [x] `decimals` defaults to `2` when omitted
- [x] Supports custom decimal places (e.g. `formatPercent(0.126, 0)` → `"13%"`)
- [x] Returns `"0.00%"` for zero
- [x] Handles negative numbers (e.g. `formatPercent(-0.1234)` → `"-12.34%"`)
- [x] Returns `"NaN%"` for `NaN`, `"Infinity%"` / `"-Infinity%"` (or string literals) for ±Infinity
- [x] Exported from `src/index.ts`
- [x] Tests pass in `tests/formatting.test.ts`

## Implementation Notes
- Guard NaN/±Infinity → return string literal
- Multiply input by 100, call `toFixed(decimals ?? 2)`, append `"%"`
