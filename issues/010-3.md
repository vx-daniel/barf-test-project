## Description
Implement the `formatCurrency(n, currency?)` number formatting utility that formats a number as a currency string, and write its tests.

## Acceptance Criteria
- [ ] `formatCurrency(n, currency?)` formats a number as a currency string (e.g. `1234.5` → `"$1,234.50"`)
- [ ] `currency` defaults to `"USD"` when omitted
- [ ] Supports common currencies: `USD` → `$`, `EUR` → `€`, `GBP` → `£`, `JPY` → `¥`
- [ ] Throws on unknown currency codes (e.g. `formatCurrency(10, 'XYZ')` → throws)
- [ ] Handles negative numbers (e.g. `formatCurrency(-1234, 'USD')` → `"-$1,234.00"`)
- [ ] Returns `"NaN"`, `"Infinity"`, `"-Infinity"` for those special values
- [ ] Exported from `src/index.ts`
- [ ] Tests pass in `tests/formatting.test.ts`

## Implementation Notes
- Lookup currency symbol from a small map; throw on unknown code
- Format absolute value with 2 decimal places using `toFixed(2)`
- Apply comma formatting to integer part (can reuse `formatWithCommas` logic internally — depends on 010-1)
- Prepend currency symbol; handle negative sign (e.g. `"-$1,234.50"`)
- Guard NaN/±Infinity first

## Dependencies
- Depends on `formatWithCommas` being available (010-1)
