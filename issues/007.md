Add `ltrim` and `rtrim` functions to `src/index.ts`.

## Functions

**`ltrim(s: string, chars?: string): string`**
Strips leading whitespace (default) or leading occurrences of any character in `chars`.

**`rtrim(s: string, chars?: string): string`**
Strips trailing whitespace (default) or trailing occurrences of any character in `chars`.

Examples:
```
ltrim('  hello')         → 'hello'
ltrim('--hello', '-')    → 'hello'
rtrim('hello  ')         → 'hello'
rtrim('hello--', '-')    → 'hello'
```

## Acceptance Criteria

- [ ] `ltrim` removes leading whitespace by default
- [ ] `ltrim` removes only the specified chars when `chars` is provided
- [ ] `rtrim` removes trailing whitespace by default
- [ ] `rtrim` removes only the specified chars when `chars` is provided
- [ ] Both return the original string if nothing to strip
- [ ] Both return `''` when the entire string is stripped
