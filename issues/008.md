Add a `truncate` function to `src/index.ts`.

## Function

**`truncate(s: string, maxLen: number, suffix?: string): string`**
Truncates `s` to at most `maxLen` characters. If truncated, appends `suffix` (default `'...'`). The total length including the suffix never exceeds `maxLen`.

Examples:
```
truncate('hello world', 8)         → 'hello...'
truncate('hello world', 8, '…')    → 'hello w…'
truncate('hi', 10)                 → 'hi'
truncate('hello', 3, '...')        → '...'
```

## Acceptance Criteria

- [ ] Returns original string when `s.length <= maxLen`
- [ ] Truncates and appends suffix when over limit
- [ ] Total result length does not exceed `maxLen`
- [ ] Default suffix is `'...'`
- [ ] Custom suffix is supported
- [ ] Handles `maxLen` shorter than suffix length gracefully
- [ ] Exported from `src/index.ts`
- [ ] Tests in `tests/truncate.test.ts`
