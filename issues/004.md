Add a `repeat` function to `src/index.ts`.

## Function

**`repeat(s: string, n: number): string`**
Returns the string repeated `n` times. If `n <= 0`, returns `''`.

Examples:
```
repeat('ab', 3)  → 'ababab'
repeat('x', 0)   → ''
repeat('x', 1)   → 'x'
```

## Acceptance Criteria

- [ ] Returns the string repeated `n` times
- [ ] `repeat(s, 0)` returns `''`
- [ ] `repeat(s, 1)` returns `s` unchanged
- [ ] `repeat('', n)` returns `''`
- [ ] Negative `n` returns `''`
- [ ] Exported from `src/index.ts`
- [ ] Tests in `tests/repeat.test.ts`
